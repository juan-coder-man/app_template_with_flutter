# Documentación para Google Play

Esta carpeta agrupa plantillas y referencias para publicar la aplicación en Google Play. Todo el contenido es **ejemplo** y debe sustituirse por textos revisados antes de publicar.

| Documento                                                        | Uso                                                                                                      |
| ---------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| [README-signing-and-build.md](README-signing-and-build.md)       | Guía de firma Android (keystore + Gradle), build de AAB/APK y checklist de renombre por plataforma.      |
| [README-google-play-versions.md](README-google-play-versions.md) | Relación entre `version` en `pubspec.yaml` y los campos de versión en Play Console.                      |
| [README-graphics.md](README-graphics.md)                         | Activos gráficos: icono, gráfico destacado, capturas; tamaños, formatos y obligatoriedad por plataforma. |
| [README-store-listing.md](README-store-listing.md)               | Nombre de la app (máx. 30 caracteres) y descripciones breve (80) y completa (4000) por idioma.           |
| [README-privacy-policy.md](README-privacy-policy.md)             | URL, campos y convención de nombres para la política de privacidad.                                      |
| [README-release-notes.md](README-release-notes.md)               | Novedades del release con bloques por localización, listo para copiar.                                   |

Estos archivos se versionan con el repositorio para mantener alineación entre builds, notas de publicación y ficha de tienda.
