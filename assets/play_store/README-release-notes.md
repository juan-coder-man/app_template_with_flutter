# Novedades del release (Google Play)

Copia el bloque siguiente **completo** y sustituye los textos de ejemplo por cambios reales. Las etiquetas `<locale>` siguen la convención habitual de Android/Google Play para distinguir idiomas y variantes regionales.

Si pegas solo en la interfaz web de Play Console, puede bastar con pegar el contenido que corresponda a cada idioma en cada campo localizado.

---
